# 在线加密文件解密SAAS网站需求文档

## 1. 产品概述

### 1.1 产品定位
- 一款面向个人和小型企业的在线加密文件解密服务平台
- 提供多种密码破解方案，满足不同用户的需求
- 无需安装软件，通过浏览器即可使用
- 支持多种加密文件格式

### 1.2 目标用户
- 忘记密码的普通用户
- 需要解密文件的小型企业
- IT技术人员
- 数据恢复从业人员

### 1.3 核心价值
- 简单易用：无需专业知识，一键上传即可发起破解
- 多种方案：提供不同级别的破解服务，满足不同需求
- 安全可靠：文件加密传输，保护用户隐私
- 高效快速：部分方案支持快速破解

## 5. 合规性与法律要求

### 5.1 用户身份验证（KYC）
- **基础验证**：邮箱验证（必填）
- **高级验证**：专业破解服务需要身份验证
  - 个人用户：身份证件验证
  - 企业用户：营业执照验证
- **验证时机**：首次使用专业破解服务时

### 5.2 文件所有权验证
- 用户需声明拥有文件的所有权或解密权限
- 系统记录用户声明和IP地址
- 可疑文件需要人工审核
- 企业用户需提供额外的授权证明

### 5.3 滥用检测与防范
- **速率限制**：
  - 匿名用户：每天最多3次免费尝试
  - 注册用户：每天最多10次免费尝试
  - 付费用户：根据会员等级设置限制
- **异常行为检测**：
  - 短时间内大量提交相似文件
  - 使用多个邮箱账号规避限制
  - 提交明显非用户所有的文件
- **风控措施**：
  - 可疑账号冻结机制
  - IP黑名单管理
  - 机器行为检测

### 5.4 合规审查流程
- **自动审查**：敏感文件名、文件类型检测
- **人工审核**：专业破解服务前的二次确认
- **审计日志**：记录所有敏感操作，保存至少6个月
- **执法配合**：设置专门的执法请求处理流程

### 5.5 法律声明
- **用户协议**：明确禁止破解非法文件
- **使用场景限制**：仅限恢复自己的文件
- **免责声明**：不对用户使用方式负责
- **数据保留**：配合执法的数据保留政策

## 8. 技术细节

### 8.1 安全设计
- **传输安全**：
  - 强制HTTPS（TLS 1.3）
  - 证书固定（Certificate Pinning）
  - HSTS启用
- **文件存储安全**：
  - 上传文件加密存储（AES-256）
  - 定期清理过期文件（成功后7天，失败后3天）
  - 严格的访问控制和权限管理
  - 文件病毒扫描（ClamAV）
- **数据安全**：
  - 用户密码bcrypt加盐哈希存储
  - 敏感数据（如密码）使用AES-256加密存储
  - 数据库连接加密
  - 定期数据备份（每日全量+实时增量）
- **API安全**：
  - API速率限制（每用户每分钟最多60次请求）
  - JWT Token过期时间控制（访问token 2小时，刷新token 7天）
  - API密钥管理和轮换
  - CORS配置
- **防护措施**：
  - 防止DDoS攻击：使用CDN或专业的DDoS防护服务
  - 防止SQL注入：使用ORM参数化查询
  - 防止XSS：输入验证和输出编码
  - 防止CSRF：Token验证
  - 依赖项安全扫描

### 8.2 性能优化
- **前端性能**：
  - 代码分割和懒加载
  - 预加载关键资源
  - 图片压缩和WebP格式
  - CDN加速静态资源
  - Gzip/Brotli压缩
- **后端性能**：
  - 数据库索引优化
  - Redis缓存策略（热点数据缓存5分钟）
  - 异步处理（Celery任务队列）
  - 数据库连接池
  - 查询结果缓存
- **破解引擎性能**：
  - 分布式计算（多节点并行）
  - GPU加速（CUDA支持）
  - 算法优化（字典生成、规则优化）
  - 结果缓存（避免重复破解）

  - 办公文档：PDF、Word（DOC/DOCX）、Excel（XLS/XLSX）、PPT（PPT/PPTX）
  - 其他：AES加密文件等

### 8.6 数据保留策略
| 数据类型 | 保留期限 | 说明 |
|---------|---------|------|
| 上传文件 | 成功后7天/失败后3天 | 自动删除 |
| 破解结果 | 永久 | 用户可主动删除 |
| 任务记录 | 180天 | 用于审计和统计 |
| 用户日志 | 90天 | 访问日志 |
| 审计日志 | 180天 | 敏感操作记录 |
| 支付记录 | 5年 | 财务合规要求 |

### 8.7 隐私保护
- **数据收集**：仅收集必要信息
- **用户权利**：
  - 数据查看权
  - 数据更正权
  - 数据删除权（被遗忘权）
  - 数据导出权
- **隐私设计**：
  - 默认隐私保护
  - 最小权限原则
  - 定期隐私审计
- **合规性**：符合GDPR、CCPA等隐私法规要求
